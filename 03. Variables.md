La convención para poner nombre a las variables es el <font style="color:#08f; font-weight:bold">camelCase</font>.
# Tipos de variables
## var
- Tipo de variable poco recomendable debido a su alcance. Una variable declarada con <font style="color:#08f; font-weight:bold">var</font> dentro de una estructura, sigue siendo accesible desde fuera de la estructura (alcance de función o global).
<font style="color:green; font-weight:bold">Ejemplo:</div>
- Si se declara dentro de una estructura como <font style="color:#08f; font-weight:bold">if</font>, sigue siendo accesible desde fuera:
```javascript
if (true) {
var edad = 30; // Declaración y asignación de valor a variable edad
}
console.log(edad); // Imprime 30
```
- Se puede modificar el valor de la variable:
```javascript
var edad = 30;
edad = 31; // modificamos el valor de la variable
```
## let
- Se consigue un mayor control de su vida útil porque se alcance se limite al bloque.
```javascript
if (true) {
let edad = 30; // Declaración y asignación de valor a variable edad
}
console.log(edad); // Eror: edad no está definida fuera del bloque
```
- Se puede modificar el valor de la variable:
```javascript
let edad = 30;
edad = 31; // modificamos el valor de la variable
```
## const
- No pueden ser reasignadas después de su inicialización (valor inmutable).
- No se puede modificar el valor:
```javascript
const persona = "Juan"
persona = "Pepe"; // Lanza TypeError de sintax
```
- Sí se puede modificar el valor de las propiedades:
```javascript
const persona = {
nombre: "Juan"; // Se le asigna a la propiedad nombre el valor "Juan"
}
persona.nombre = "Pepe"; // Se le asigna a la propiedad nombre el valor "Pepe"
```
