- Los tipos de datos se asignan dinámicamente (tipado dinámico), por lo que no hay que establecerlos.
- Se dividen entre los tipos <font style="color:#08f; font-weight:bold">primitivos</font> y de <font style="color:#08f; font-weight:bold">de referencia</font>.
- Los **tipos primitivos** se almacenan en el tipo de memoria <font style="color:#08f; font-weight:bold">Stack</font> y los **de referencia** en el tipo de memoria <font style="color:#08f; font-weight:bold">Heap</div>:
	- <font style="color:#08f; font-weight:bold">Stack</font>:
		- El espacio esta asignado de forma fija, por lo que no se puede aumentar.
	- <font style="color:#08f; font-weight:bold">Heap</font>:
		- El espacio puede cambiar de manera dinámica.

| Primitivos | De referencia |
| ---------- | ------------- |
| Number     | Array         |
| String     | Object        |
| Boolean    | Functions     |
| Undefined  | Clases        |
| Null       |               |
# Datos primitivos
## Number
```javascript
let numero = 1;
```
## String
```javascript
let texto = "Hola mundo";
```
## Boolean
```javascript
let verdadero = true;
let falso = false;
```
## Undefined
```javascript
let noDefinido; // El valor por defecto de las variables

```

```javascript

```
# De referencia
## Object
- Colección de <font style="color:#08f; font-weight:bold">propiedades</font>.
- Cada propiedad es una asociación clave-valor.
- Las propiedades se separan mediante comas (**,**).
### Creación de objetos
```javascript
let persona = {
nombre: "Pepe",
edad: 34, // La fila completa se denominta par llave-valor, lo de la izquierda es llave y lo de la derecha valor. En conjunto es una propiedad
};
```
### Acceso a los valores de las propiedades de los objetos
- Se puede acceder al valor de una <font style="color:#08f; font-weight:bold">propiedad</font> a través de 2 maneras:
	- Mediante punto "<font style="color:black; font-weight:bolder">.</font>":
```javascript
console.log(persona.nombre); // Imprime el valor de la propiedad nombre
```
- Mediante paréntesis cuadrados "<font style="color:black; font-weight:bolder">[]</font>":
```javascript
console.log(persona['nombre']); // Imprime el valor de la propiedad nombre
```
### Modificación de los valores de las propiedades de los objetos
- Se pueden modificar el valor de las <font style="color:#08f; font-weight:bold">propiedades</font> de un objeto.
```javascript
persona.nombre = "Ana"; // Modificación a traves del punto
persona['nombre'] = "Laura";
```
### Eliminar propiedades
- Se puede eliminar una propiedad con la palabra clave <font style="color:#08f; font-weight:bold">delete</font>:
```javascript
delete persona.nombre; // Elimina la propiedad nombre del objeto persona
```
## Array
- Lista ordenada de elementos.
### Crear un array
- Crear array <font style="color:orange; font-weight:bold">vacío</font>:
```javascript
let animales = [];
```
- Crear array <font style="color:orange; font-weight:bold">con elementos</font>:
```javascript
let animales = ['perro', 'gato', 'ratón'];
```
### Acceder a los elementos de un array
- Para acceder a <font style="color:orange; font-weight:bold">un único elemento</font>, se utilizará el índice del elemento de la colección escrito entre paréntesis cuadrados "<font style="color:black; font-weight:bolder">[]</font>" en el que el primer elemento se encuentra en el índice 0:
```javascript
console.log(animales[0]); // Imprime al elemento "perro"
```
- Para acceder a <font style="color:orange; font-weight:bold">todos los elementos</font> del array , no se utilizará el índice:
```javascript
console.log(animales); // Imprime todos los elementos del array.
```
### Modificar el valor de los elementos de un array
- Se puede modificar el valor de un elemento de un array apuntando a su índice:
```javascript
animales[1] = 'tigre'; // modificaria el valor de gato a tigre
```
### Consultar el número de elementos del array
- Con el método <font style="color:#08f; font-weight:bold">length</font> podemos consultar el número de elementos que componen el array:
```javascript
console.log(animales.length);
```
## Funciones
- Se utiliza la palabra reservada <font style="color:#08f; font-weight:bold">function</font>:
```javascript
/*
Función SIN return
*/
function saludar() {
console.log('Hola Mundo');
}

// Ejecutar función
saludar(); // Imprimira en consola "Hola Mundo".

/*
Función CON return
*/
function sumar() {
return 3 + 3;
}

// Ejecutar función
let resultado = sumar();
console.log = resultado;
```
### Uso de argumentos y parámetros
- Podemos asignar parámetros en las funciones para que estos reciban argumentos
```javascript
// Función con PARÁMETROS
function sumar (num1, num2) {
return num1 + num2;
}

// Ejecutar la función con ARGUMENTOS.
let resultadoSuma = sumar(3, 3);
console.log = resultadoSuma;
```
# Consultar el tipo de dato
Con la función <font style="color:salmon; font-weight:bold">typeof</div> se pude consultar el tipo de dato.
```javascript
let texto = "Hola mundo";
console.log(typeof texto); // Imprimirá string
```
<font style="color:red; font-weight:bold">Excepción</div>:
El tipo de dato <font style="color:#08f; font-weight:bold">null</div> devuelve object:
```javascript
let nulo = null;
console.log(typeof nulo); // Imprimirá object porque representa un objeto vacío
```
