- Los tipos de datos se asignan dinámicamente (tipado dinámico), por lo que no hay que establecerlos.
- Se dividen entre los tipos <font style="color:#08f; font-weight:bold">primitivos</div> y de <font style="color:#08f; font-weight:bold">de referencia</div>.
- Los **tipos primitivos** se almacenan en el tipo de memoria <font style="color:#08f; font-weight:bold">Stack</div> y los **de referencia** en el tipo de memoria <font style="color:#08f; font-weight:bold">Heap</div>:
	- <font style="color:#08f; font-weight:bold">Stack</div>:
		- El espacio esta asignado de forma fija, por lo que no se puede aumentar.
	- <font style="color:#08f; font-weight:bold">Heap</div>:
		- El espacio puede cambiar de manera dinámica.

| Primitivos | De referencia |
| ---------- | ------------- |
| Number     | Array         |
| String     | Object        |
| Boolean    | Functions     |
| Undefined  | Clases        |
| Null       |               |
# Datos primitivos
## Number
```javascript
let numero = 1;
```
## String
```javascript
let texto = "Hola mundo";
```
## Boolean
```javascript
let verdadero = true;
let falso = false;
```
## Undefined
```javascript
let noDefinido; // El valor por defecto de las variables
let undef = undefined; // Podemos establecer el valor undefined
```
## Null
```javascript
let nulo = null; // cuando queremos establecer el menu como nulo
```
# De referencia
## Object
- Colección de propiedades.
- Cada propiedad es una asociación clave-valor.
- Las propiedades se separan mediante comas (**,**).
```javascript
let persona = {
nombre: "Pepe",
edad: 34
};
```
## Array
- Lista ordenada de elementos.

# Consultar el tipo de dato
Con la función <font style="color:salmon; font-weight:bold">typeof</div> se pude consultar el tipo de dato.
```javascript
let texto = "Hola mundo";
console.log(typeof texto); // Imprimirá string
```
<font style="color:red; font-weight:bold">Excepción</div>:
El tipo de dato <font style="color:#08f; font-weight:bold">null</div> devuelve object:
```javascript
let nulo = null;
console.log(typeof nulo); // Imprimirá object porque representa un objeto vacío
```
