Hay dos tipos de coversiones:
a) <font style="color:#08f; font-weight:bold">Implícitas</font>:
- Son aquellas que se realizan de manera automática.
- Con el operador suma "<font style="color:#08f; font-weight:bold">+</font>", si hay una cadena de texto, siempre se convierte a texto y se concatena:
	- <font style="color:green; font-weight:bold">Ejemplo</div>:
```javascript
let suma = 3 + '3';
console.log(suma); // Imprime 33 porque el 3 se convierte en cadena y concatena con '3'
```
- Con los otros operadores aritméticos, "<font style="color:#08f; font-weight:bold">-</font>", "<font style="color:#08f; font-weight:bold">*</font>", "<font style="color:#08f; font-weight:bold">/</font>", "<font style="color:#08f; font-weight:bold">%</font>" se prioriza la operación matemática:
	-  <font style="color:green; font-weight:bold">Ejemplo</div>:
```javascript
let rest = 3 - '2';
console.log(suma); // Imprime 1 porque el 2 se convierte en numero.
```
b) <font style="color:#08f; font-weight:bold">Explícitas</font>:
- Se realizan de manera manual mediante métodos o funciones.
	-  <font style="color:green; font-weight:bold">Ejemplo</div>:
```javascript
// Conversión a Number
let cadena = '123';
let numero = Number(cadena);
console.log(typeof numero); // Imprime "number"

// Conversión a String
let valorNumerico = 456;
let cadenaTexto = String(valorNumerico);
console.log(cadenaTexto); // Imprime "string"

// Conversión Boolean
let numeroFalse = 0;
let valorBoolean = Boolean(numeroFalse);
console.log(valorBoolean); // Imprime "false"
```
