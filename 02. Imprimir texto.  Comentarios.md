# Imprimir texto
La función para imprimir texto es console.log("texto").
- <font style="color:green; font-weight:bold">Ejemplos</div>:
- Imprimir texto de forma literal:
```javascript
console.log("texto a imprimir");
```
- Imprimir variable:
```javascript
console.log("nombreVariable");
```
# Comentarios
Se puede imprimir comentarios de una línea y de múltiples líneas
```javascript
// Comentario de una línea
console.log("nombreVariable");

/*
Este es un comentario
de muchas
líneas
*/
```
